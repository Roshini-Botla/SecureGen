# SecureGen 
---

## 🚀 Usage

1. Open **http://localhost:8000**
2. Upload any AI-generated code file (`.py`, `.js`, `.java`, `.cpp`, etc.)
3. Click **Analyze for Vulnerabilities**
4. View the detailed security report with CWE IDs, severity levels, and fix suggestions

API documentation available at **http://localhost:8000/docs**

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js, HTML5, CSS3 |
| Backend | Python, FastAPI, Uvicorn |
| AI Engine | Anthropic Claude API |
| Security Standards | CWE, OWASP Top 10 |
| Communication | REST API |
